## 🇺🇸 What is Vault? | 🇪🇸 Qué es Vault?

🇺🇸 Vault is a secret manager to store and manage many different kind of secrets (key pair, certificates, database credentials, kubernetes secrets, etc). Also provide several authentication methods depending on your requirements, these could be authenticate to the web UI with your O365 credentials or with your Kubernetes access account, this will depend on your use case. Additionally Vault can be used to create/store credentials or retrieve them with many programming languages, it is integrated with Terraform and can be used with CI/CD pipelines, I personally use it with Gitlab CI/CD.

 🇪🇸 Vault es un gestor de secretos que almacena y gestiona diversos tipos de secretos (pares de claves, certificados, credenciales de base de datos, secretos de Kubernetes, etc.). También ofrece varios métodos de autenticación según sus necesidades, como la autenticación en la interfaz web con sus credenciales de O365 o con su cuenta de acceso de Kubernetes, según su caso de uso. Además, Vault permite crear, almacenar o recuperar credenciales con diversos lenguajes de programación. Está integrado con Terraform y se puede usar con pipelines de CI/CD. Personalmente, lo uso con Gitlab CI/CD.
